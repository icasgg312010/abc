# 🧬 Análisis Epidemiológico: Resurgimiento del Sarampión y Brechas de Vacunación

## 📌 Descripción del Proyecto
Este proyecto analiza la dinámica actual del sarampión a nivel global, con un enfoque crítico en la situación epidemiológica de **México (2020-2025)**. Utilizando herramientas de ciencia de datos, se explora la correlación entre la caída de las tasas de inmunización y la aparición de brotes en nodos urbanos.

### 🩺 Contexto Médico
El sarampión posee un $R_0$ de 12-18, lo que exige una cobertura vacunal (MCV1/MCV2) del **95%** para garantizar la inmunidad de rebaño. Este análisis identifica los puntos de quiebre donde la salud pública se vuelve vulnerable.

## 🚀 Características Principales
*   **Dashboard de Riesgo Animado:** Visualización de la relación inversa entre cobertura vacunal y casos reportados (2000-2024).
*   **Mapa de Calor Global:** Identificación de "Hotspots" y rutas de importación viral.
*   **Análisis de Series de Tiempo:** Monitoreo de los brotes en México (CDMX/EdoMex 2020 y Alerta 2025).
